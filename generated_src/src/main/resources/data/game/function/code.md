# 개요
- 당신은 다음 프롬포트에 따라 무한 모드를 개발해야 합니다.

# 설명
- game/function 안에 infinity라는 디렉터리를 새로 만들고, 그곳에 기존 클래식 모드와는 다른 무한 모드를 제작해야 됩니다.
- 무한 모드는 일반적인 라운드 웨이브를 따라가지 않으며, 자체적인 알고리즘을 사용합니다.
- 무한 모드를 위해 설정된 몹 리스트가 존재하며, 웨이브가 진행됨에 따라 해당 몹이 나오게 됩니다.
- 해당 리스트는 api/map/data/맵 이름/infinity에 존재합니다.
- 몹은 라운드가 진행됨에 따라 데이터 순서대로 등장하게 되며, 다음 규칙을 따릅니다:
	1. 일반 몹의 경우, 첫 등장 시 5마리가 1초 간격을 두고 등장, 엘리트 몹은 첫 등장 시 2마리가 5초 간격을 두고 등장, 보스 몹의 경우 1마리가 등장합니다.
	2. 일반 몹의 경우에는 이후 라운드마다 등장하는 마릿수가 1마리씩, 엘리트 몹은 2라운드에 1마리씩, 보스 몹은 15라운드에 1마리씩 증가합니다. 단, 보스 몹은 5라운드마다 등장합니다. 만약 등장 보스 몹이 5마리가 되면, 매 라운드마다 1마리씩 등장하는 것으로 변경되고 더 이상 증가하지 않습니다.
	3. 일반 몹의 경우에는 첫 등장 이후 7라운드가 지나면 더 이상 등장하지 않습니다. 엘리트 몹은 첫 등장 이후 15라운드가 지나면 더 이상 등장하지 않습니다. 보스 몹의 경우에는 첫 등장 이후 25라운드가 지나면 더 이상 증가하지 않습니다. (등장은 합니다.)
	4. 3라운드마다 새로운 몹이 등장합니다. 단, 1라운드는 예외입니다. (1, 3, 6, 9, 12, 15, 18, 21, 24, 27 ... 라운드에 새로운 몹이 등장)
- 무한 모드는 라운드가 진행됨에 따라 특수한 규칙을 따릅니다:
	1. 몹의 체력은 추가 보정을 받아 적용됩니다. 1.04^(라운드) 만큼의 체력을 지닙니다.
	2. 몹의 이동 속도 역시 추가 보정을 받습니다. 1.005^(라운드) 만큼의 이동 속도를 지닙니다.
	3. 몹의 방어력 역시 추가 보정을 받습니다. 5라운드마다 방어력 3이 상승합니다.
